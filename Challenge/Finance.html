Hi there, This is a single page application to draw the spending habits of a person using D3.js from transaction data in a CSV file and make small suggestions.
This application uses JQuery, Bootstrap, Papaparse for Parsing, Angular, the Google Places API and D3 for the graph. (Original csv data provided by Intuit Inc. and used only for demonstration purposes)

A sample CSV can be seen by clicking that button in the navBar. The app expects at least 3 months of data. Ideally, intended for 2 years worth of data.

Instructions: Click "Person A" or "Person B" to show their spending habits for 2 years provided, or use your own CSV file and then click "Use File!". The app stores new data every time it queries unseen keys(transactions) from new csv files using the Google Places API, the app already contains the data derived from Person A and Person B who are from Indianapolis.
